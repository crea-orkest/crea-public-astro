# CREA Orkest public Astro site

The CREA Orkest Astro front-end. Build with Astro, content from DatoCMS and images from Cloudinary.

## Production

The website is updated by adding a new version tag. When adding the new version tag the `production` branch will be updated. This triggers a production build in Cloudflare.
